  • Make Mediawiki configuration more flexible. List of open issues.
  • Improve the usability of backup commands. List of open issues.
  • Improve some of the UBOS tools. List of open issues.
  • Improve the core UBOS administration commands. List of open issues.
  • New apps and other packages people have requested to run on UBOS. Here's the list. This involves packaging up the app and testing it according to the instructions in the developer's guide.
  • Bring some more useful Wordpress plugins to UBOS. Here are some examples for how to do that.

... or whatever else you think would be useful.

Per new Github convention, some of the issues are marked with the tag "good first issue"". These are ones expected to be easier to solve for people who are new to UBOS.
