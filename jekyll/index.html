Let's keep control over our data.

UBOS is a new Linux distro that aims to make it 10x easier to run server-side apps for yourself, or for your family.

Why give your valuable data to Google or some other data overlord, if you can keep it under your own control? All you need is a spare (physical, virtual, or cloud) computer that can run Linux. UBOS automates much of the administration, so you can get on with life.

Want to use ownCloud instead of Dropbox? Shaarli instead of delicious? An RSS reader or an open-source Internet-of-Things app? UBOS makes that easy and quick.

HOWTO

Run UBOS

Download an x86 image and write it to a USB boot stick, or:

Download an VirtualBox image and use it as your main virtual hard drive.

Install an app

# ubos-admin createsite
App: owncloud
Host: family.home.com

Code installed, database provisioned, virtual host configured...

Upgrade everything

# ubos-admin update
Done.

OS updated, apps upgraded, database migrations performed, services restarted...

Backups

# ubos-admin backup \
  --out all.ubos-backup
# ubos-admin restore \
  --in all.ubos-backup

Plenty of options for more fine-grained control.
