Get the image

  • To run in VirtualBox: download this image, uncompress, and use as the main virtual hard drive in VirtualBox.
  • To boot UBOS from a USB stick: download this image, uncompress, and write to a USB stick in "raw" format.

For a detailed series of steps, refer to the Installation section in the UBOS user documentation.

Install one or more apps

Once booted, log in from the console as root. To install ownCloud:

# ubos-admin createsite

and answer the questions. Use owncloud as the name of the app.

To show installed apps and at which virtual hosts they run:

# ubos-admin listsites

To remove an app:

# ubos-admin undeploy --host hostname

To back up all data from your apps:

# ubos-admin backup --out all.ubos-backup

For more details, refer to the UBOS user documentation.
